How to Register a User Account

To register an account, you can follow the method below.

  • click register below HomePage
  • Enter Username (Full name as per ID card)
  • Enter Number Phone active
  • Enter Password Consisting of Letters and Numbers for example: (Xxx123@)
  • Repeat the password that you have created then click REGISTER.
  • Enter Verification code which has been sent via SMS and click NEXT.

    • If there is no incoming SMS, try Resending. Wait until there is information Verification Successful.
